# 🧠 MindOS AI — Personal Operating System for Your Life

MindOS AI is an AI-powered personal operating system that helps you **think, plan, and act intelligently** by unifying notes, tasks, goals, and conversations into a single platform.

> Not just a productivity app — a system that understands you.

---

## 🚀 Features

### 🧠 Memory Engine
- Stores notes, tasks, goals, and conversations
- Semantic search to recall past information
- Context-aware linking of ideas

### 💬 AI Thinking Partner
- Chat with AI using personal context
- Get help with decisions, planning, and problem-solving

### 🎯 Goal Intelligence System
- Breaks goals into actionable tasks
- Tracks progress automatically

### 📅 Smart Planner
- Auto-schedules tasks
- Dynamically adjusts based on user behavior

### 📊 Productivity Dashboard
- Task completion analytics
- Goal tracking
- Visual insights (charts, progress bars)

### 🎙️ Voice Transcription
- Convert voice → text
- Hands-free note-taking and AI interaction

### 🧩 Context-Aware Suggestions
- AI suggests what to do next
- Based on goals, tasks, and past activity

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios
- React Router
- Framer Motion

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### AI / ML
- GROQ API (AI reasoning & chat)
- Embeddings (semantic search)
- Vector DB (optional)

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/mindos-ai.git
cd mindos-ai

*Setup Backend*
cd backend
npm install

Create .env file:

PORT=5000
MONGO_URI=your_mongodb_url
GROQ_API_KEY=your_api_key

Run backend:

npm run dev

*Setup Frontend*

cd frontend
npm install
npm run dev
